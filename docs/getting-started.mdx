# Getting Started

This guide will help you quickly integrate Headless Calendar into your project. While the core library is framework-agnostic, we'll demonstrate a basic setup using a popular framework (e.g., React) to illustrate the workflow. Remember that detailed instructions for each framework are available in the "Frameworks" section.

## 1. Installation

First, install the core `headless-calendar` wrapper for your chosen framework.

<MarkdownTabs>
  <Tab label="React">
    ```bash
    # Using npm
    npm install @verbpatch/react-calendar

    # Using yarn
    yarn add @verbpatch/react-calendar

    # Using pnpm
    pnpm add @verbpatch/react-calendar
    ```
  </Tab>
  <Tab label="Vue.js" remoteUrl="Vue/basic">

</MarkdownTabs>
