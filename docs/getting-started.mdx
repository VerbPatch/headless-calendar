# Getting Started

This guide will help you quickly integrate Headless Calendar into your project. While the core library is framework-agnostic, we'll demonstrate a basic setup using each framework we support to illustrate the workflow. Remember that detailed instructions for each framework are available in the "Frameworks" section.

## 1. Installation

First, install the wrapper for your chosen framework.

<MarkdownTabs library="calendar">
  <Tab active={true} label="React" icon={<SiReact />}>
    ```bash
    # Using npm
    npm install @verbpatch/react-calendar

    # Using yarn
    yarn add @verbpatch/react-calendar

    # Using pnpm
    pnpm add @verbpatch/react-calendar
    ```
  </Tab>
  <Tab active={false} label="Vue.js" icon={<SiVuedotjs />}>
    ```bash
    # Using npm
    npm install @verbpatch/vuejs-calendar

    # Using yarn
    yarn add @verbpatch/vuejs-calendar

    # Using pnpm
    pnpm add @verbpatch/vuejs-calendar
    ```
  </Tab>
  <Tab active={false} label="Svelte" icon={<SiSvelte />}>
    ```bash
    # Using npm
    npm install @verbpatch/svelte-calendar

    # Using yarn
    yarn add @verbpatch/svelte-calendar

    # Using pnpm
    pnpm add @verbpatch/svelte-calendar
    ```
  </Tab>
</MarkdownTabs>
